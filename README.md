# Maison Lumière

A luxury fashion house showcase website inspired by soft feminine European couture aesthetics.

Maison Lumière is a fully responsive portfolio project featuring a refined editorial layout, a full-screen campaign hero section, and a dynamic light/dark theme system with persistent user preference.

---

##  Project Overview

Maison Lumière represents a high-end fashion house specializing in Parfums and Jewellery.  

This project was designed to demonstrate the balance between elegant luxury branding and clean, scalable front-end development.

The visual direction is inspired by:
- Soft feminine luxury
- Editorial fragrance campaigns
- Champagne gold accents
- Linen textures
- Minimal European couture design

---

## 🌗 Key Features

- Fully responsive design (mobile + desktop optimized)
- Full-screen editorial hero section
- Light / Dark theme toggle
- Persistent theme preference using `localStorage`
- Soft textured UI surfaces
- Elegant serif typography system
- Luxury-focused layout composition
- Clean navigation with mobile menu
- Portfolio-ready structure

---

## 🎨 Design System

### Light Theme
- Background: Soft Ivory `#F4EFE6`
- Sections: Warm Cream `#EFE6D8`
- Text: Deep Charcoal `#1C1C1C`
- Accent: Champagne Gold `#C6A75E`

### Dark Theme
- Background: Matte Black `#0E0E0E`
- Sections: Deep Charcoal `#1C1C1C`
- Text: Soft Ivory `#F4EFE6`
- Accent: Champagne Gold `#C6A75E`

Gold elements are styled to feel metallic and refined rather than flat.
